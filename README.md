# Дрикинт бот для списаний
Данный бот создан для подсчёта списаний в одной из кофеен дринкита. Я не являюсь сотрудником дринкита. Данный проект является некоммерческим, делается для моих друзей бариста.
Для его работы вам необходимо создать файл .env в основной папке бота, а в него написать BOTTOKEN="(сюда необходимо записать ваш ключ тг бота)"
